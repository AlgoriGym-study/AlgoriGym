# 📌 AlgoriGym

## 🛠️ 스터디 규칙
1. **두명씩 돌아가면서 공통문제 출제**
   - 화요일까지 각 담당자들은 이슈에 공통 문제 출제 
   - 1번 : 박재은
   - 2번 : 추수연
   - 3번 : 서재곤
   - 4번 : 윤현석
   - 5번 : 이영우
   - 6번 : 김미림
   - 7번 : 김선민민
2. **공통 과제 풀이 후 코드 리뷰**
   - 문제 풀이 후 화요일까지 깃허브에 업로드 (꼼꼼한 주석 작성)
   - 수요일 미팅 전까지 코드 리뷰 진행 (꼼꼼한 코드 리뷰 필수!)
3. **미팅**
   - 알고리즘 손코딩(30분)
   - 공통 과제 코드 리뷰(1시간)
4. **공통 문제 복습**
   - 공통 문제 풀이 깃허브 제출 화요일까지
